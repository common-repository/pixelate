Replaces any image with a pixelated version in a canvas element  and optionally reveal them on hover/click..

== Description ==

This is a port from pixelate.js by 43081j.

It replaces any selected image with a pixelated version in a canvas and optionally reveal them on hover/click..

Just use [pixelate][/pixelate] shortcode around your image.

To reveal on click add revealonclick="true" to the shortcode ->[pixelate revealonclick="true"][/pixelate].

Reveal on hover is the default. 

== Installation ==

1. Upload `ogpixelated` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
